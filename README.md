
#

foodie is a web-based restaurant management and table booking system developed using the Django framework. The application allows customers to browse restaurants, reserve tables, manage bookings, and handle orders efficiently.

## 🚀 Features

- Restaurant listing and management
- Customer registration and management
- Table reservation system
- Booking management
- Payment tracking
- Coupons and discount system
- Parking slot management
- Customer chat interface
- Static media and image handling

## 🛠 Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- Bootstrap (for UI components)
